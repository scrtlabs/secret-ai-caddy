# Place your TLS certificates in this folder:
#
#   fullchain.pem  — full certificate chain (cert + intermediates)
#   privkey.pem    — private key
#
# These paths are referenced in Caddyfile:
#   tls /etc/ssl/certs/fullchain.pem /etc/ssl/private/privkey.pem
#
# Both files are gitignored (.gitignore excludes *.pem, *.crt, *.key).
